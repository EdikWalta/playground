## Technologies Used

- **Java 25**: The programming language used.
- **Spring Boot 4.0.1**: Framework for building the application.
- **Gradle 9.1.0**: Build tool for dependency management and project automation.
- **JUnit 6**: Testing framework used.
- **Next.js 16.1.3**: Framework for building the frontend application.


## Getting Started

### Prerequisites

- Java 25
- Gradle 9.1.0

### Running the Application

1. Clone the repository:
   ```bash
   git clone https://github.com/EdikWalta/playground.git
   cd playground
   ```

2. Build the project:
   ```bash
   ./gradlew build
   ```

3. Run the application:
   ```bash
   ./gradlew bootRun
   ```
